# aarontang.github.io

个人技术博客，基于 GitHub Pages 托管。

## 关于

我是 Aaron Tang，一名软件工程师，专注于后端架构与工程效能。

这里记录技术学习心得、项目实践以及对软件工程的思考。

## 技术栈

- **后端**: Java, Spring Boot
- **前端**: TypeScript, React
- **基础设施**: Docker, MySQL, Redis, Git

## 项目结构

```
/
├── index.html          # 主页
├── articles/           # 文章目录
│   └── hello-world/   # 示例文章
├── projects/           # 项目目录
│   └── demo-project/  # 示例项目
└── assets/             # 静态资源
    ├── css/           # 样式文件
    └── js/            # 脚本文件
```

## 快速开始

本地预览：

```bash
# 直接用浏览器打开
open index.html

# 或使用任意静态服务器
npx serve .
```

## 部署

推送到 `main` 分支后，GitHub Actions 会自动部署到 GitHub Pages。
