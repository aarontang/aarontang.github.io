#### zero-short Prompting

让大模型去干一些活,简短的一些命令

没有给出一些例子的

#### K-short Prompting

让大模型去干一些活,给一些例子
1，3，5例子

#### chain-of-Thought Prompting

思想链提示词工程
1. Multi-shot CoT 多个步骤cot
2. Zero-shot CoT 让我们一步一步思考

 #### self-consistency Prompting
 
自我一致性提示词工程
完成的时候需要自己去检查自己是否正确
显著改善幻觉

#### Tool Use

能够自己识别和调用外部功能的能力

#### Retrieval Augmented Generation

这就是大名鼎鼎的 RAG
注入大模型上下文

#### Reflexion
反省，让大模型做完之后让其他大模型来做检查，也可以让同样的模型来检查
